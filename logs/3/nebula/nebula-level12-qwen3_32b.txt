```lua

If the password contains shell metacharacters like `;`, `&`, `|`, or `#`, the shell will interpret these as command separators or comments, allowing the attacker to execute arbitrary commands.

For example, if the password is:

```plaintext

```

```bash

This allows the attacker to execute arbitrary commands **as the user running the server** (likely `flag12`).
